Introduction to Polynomials

Polynomials are fundamental mathematical expressions that you'll encounter frequently in pre-calculus and beyond. They are composed of variables and constants combined using addition, subtraction, and multiplication operations.

A polynomial typically has the following form:

\(P(x) = a_nx^n + a_{n-1}x^{n-1} + \ldots + a_2x^2 + a_1x + a_0\)

  • \(P(x)\) is the polynomial function.
  • \(x\) is the variable or indeterminate.
  • \(a_n, a_{n-1}, \ldots, a_2, a_1, a_0\) are coefficients (constants).
  • \(n\) is a non-negative integer, and it represents the highest degree of the polynomial.

The highest degree term, \(a_nx^n\), is called the leading term, and \(a_n\) is the leading coefficient.

Polynomials can have various degrees, and they can be used to model real-world phenomena, solve equations, and perform algebraic operations.

Let's explore polynomials further by looking at the degree of the polynomial

Classifying Polynomials by Degree

Polynomials are named and classified based on the degree of the highest power of the variable \(x\) in the polynomial expression. The degree is the exponent of the highest term.

1. Constant Polynomial: Degree \(0\)

\(P(x) = a_0\) where \(a_0\) is a constant.

2. Linear Polynomial: Degree \(1\)

\(P(x) = a_1x + a_0\) where \(a_1\) and \(a_0\) are constants and \(a_1 \neq 0\).

3. Quadratic Polynomial: Degree \(2\)

\(P(x) = a_2x^2 + a_1x + a_0\) where \(a_2\), \(a_1\), and \(a_0\) are constants and \(a_2 \neq 0\).

4. Cubic Polynomial: Degree \(3\)

\(P(x) = a_3x^3 + a_2x^2 + a_1x + a_0\) where \(a_3\), \(a_2\), \(a_1\), and \(a_0\) are constants and \(a_3 \neq 0\).

Introduction to Linear Functions

Linear equations are an important tool in science and many everyday applications. They allow scientists to describe relationships between two variables in the physical world, make predictions, calculate rates, and make conversions, among other things.

Constant Function

A constant function is a type of mathematical function where the output (dependent variable) is the same for all values of the input (independent variable). In other words, it represents a horizontal line on the coordinate plane.

- The domain of a constant function is the set of all real numbers because you can plug in any value for the independent variable. In interval notation $x \in (-\infty, \infty)$.

- The range of a constant function consists of a single value, which is the constant value of the function. In set notation, the range is expressed as \(\{c\}\), where \(c\) is the constant.

- Since a constant function has a single constant value for all inputs, it does not have a maximum or minimum value in the traditional sense. The constant itself is both the maximum and minimum.

- A constant function is symmetric with respect to the y-axis because if you reflect it across the y-axis, it remains unchanged. In other words, \(f(x) = f(-x)\) for all values of \(x\).

Linear Function

A linear function is a fundamental mathematical function that represents a straight line on the coordinate plane. It has the general form:

\(f(x) = mx + b\)

  • - The domain of a linear function is the set of all real numbers because it covers all possible values of the independent variable \(x\).
  • - The range of a linear function is also the set of all real numbers because it can produce any real number as the output.
  • - A linear function does not have a maximum or minimum value in the traditional sense because it extends infinitely in both directions.
  • - The slope (\(m\)) of a linear function determines its steepness. If \(m > 0\), the line rises as \(x\) increases, and if \(m < 0\), the line falls as \(x\) increases.
  • - The y-intercept (\(b\)) is the value where the line crosses the y-axis. It represents the output (\(f(x)\)) when \(x = 0\).
  • - A linear function is symmetric with respect to the origin (0,0) if it passes through the origin. In other words, if \(f(x) = -f(-x)\) for all values of \(x\).

Example 1: Writing Linear Functions

Suppose a maglev train were to travel a long distance, and that the train maintains a constant speed of 83 meters per second for a period of time once it is 250 meters from the station. How can we analyze the trainâ€™s distance from the station as a function of time?

The speed is the rate of change. Recall that a rate of change is a measure of how quickly the dependent variable changes with respect to the independent variable. The rate of change for this example is constant, which means that it is the same for each input value. Solving the Equation \(2(4x + 28) = 24\) for $x$

Often we want to find the value of $x$ that satisfies the equation:

\[2(4x + 28) = 24\]

Step 1: Distribute the \(2\) on the left side of the equation:

\[8x + 56 = 24\]

Step 2: Subtract \(56\) from both sides of the equation to isolate \(8x\):

\[8x = 24 - 56\]

Step 3: Simplify the right side:

\[8x = -32\]

Step 4: Finally, divide both sides by \(8\) to solve for \(x\):

\[x = \frac{-32}{8}\]

Step 5: Simplify further:

\[x = -4\]

So, the solution to the equation \(2(4x + 28) = 24\) is \(x = -4\).

Calculating the Slope

Explanation

We want to find the slope (m) of a line passing through two points, (5, 4) and (10, 8).

The formula to calculate slope is:

$$m = \frac{{y_2 - y_1}}{{x_2 - x_1}}$$

Calculation

Given points:

  • \( x_1 = 5 \)
  • \( y_1 = 4 \)
  • \( x_2 = 10 \)
  • \( y_2 = 8 \)

Plugging these values into the formula:

\( m = \frac{{8 - 4}}{{10 - 5}} \)

\( m = \frac{4}{5} \)

So, the slope (m) is \(\frac{4}{5}\).

Example 3: Finding 'b'

Given the equation \(y = 5x + b\). We want to find the value of 'b' knowing the line passes through the point (2, 1).

Substituting the Point (2, 1)

We know that the point (2, 1) lies on the line. So, we can substitute these values into the equation:

\(1 = 5 \cdot 2 + b\)

Now, we can solve for 'b'.

Solving for 'b'

\(1 = 10 + b\)

To isolate 'b', subtract 10 from both sides:

\(b = 1 - 10\)

\(b = -9\)

So, the value of 'b' is -9.

The Final Equation

Now that we've found the value of 'b', our equation becomes:

\(y = 5x - 9\)

This is the equation of the line passing through the point (2, 1).

The other two important polynomials families:

Quadratic Function

A quadratic function is a type of mathematical function that represents a parabolic curve on the coordinate plane. It has the general form:

\(f(x) = ax^2 + bx + c\)

  • - The domain of a quadratic function is the set of all real numbers because it covers all possible values of the independent variable \(x\).
  • - The range of a quadratic function depends on the leading coefficient (\(a\)). If \(a > 0\), the range is \([k, +\infty)\), where \(k\) is the minimum value of the function. If \(a < 0\), the range is \((-\infty, k]\), where \(k\) is the maximum value of the function.
  • - The maximum or minimum value of a quadratic function depends on the sign of the leading coefficient (\(a\)). If \(a > 0\), the function has a minimum value at the vertex. If \(a < 0\), the function has a maximum value at the vertex.
  • - The vertex of the parabolic curve represents the maximum or minimum point of the quadratic function. The vertex coordinates are given by \((h, k)\), where \(h\) is the x-coordinate of the vertex and \(k\) is the maximum or minimum value.
  • - A quadratic function is symmetric with respect to its axis of symmetry. The axis of symmetry is a vertical line that passes through the vertex. It is given by \(x = -\frac{b}{2a}\).

Cubic Function

A cubic function is a mathematical function that represents a curve with a single "bump" or "dip" on the coordinate plane. It has the general form:

\(f(x) = ax^3 + bx^2 + cx + d\)

  • - The domain of a cubic function is the set of all real numbers because it covers all possible values of the independent variable \(x\).
  • - The range of a cubic function depends on the leading coefficient (\(a\)). If \(a > 0\), the range is \((-\infty, +\infty)\), meaning the function can take any real value. If \(a < 0\), the range is limited, and the function has a maximum and minimum value.
  • - The maximum or minimum value of a cubic function depends on its shape and the sign of the leading coefficient (\(a\)). If \(a > 0\), the function has a minimum value at the local minimum point. If \(a < 0\), the function has a maximum value at the local maximum point.
  • - The cubic function may have one or more inflection points, where the curve changes from concave up to concave down (or vice versa). These points can affect the function's shape.
  • - The symmetry of a cubic function depends on its specific coefficients. In general, cubic functions are not symmetric, but they can exhibit different types of symmetry based on their terms and coefficients.

X-Axis, Y-Axis, and Origin Symmetry

X-Axis Symmetry

X-axis symmetry, also known as horizontal symmetry, occurs when a figure or shape is mirror-symmetric with respect to the x-axis.

This means that if a point \((x, y)\) is part of the shape, then the point \((x, -y)\) must also be part of the shape.

For example, the letter "H" exhibits x-axis symmetry because it looks the same when flipped upside down along the x-axis.

A circle is an example of x-axis symmetry because if we fold along the x-axis each part will fall perfectly ontop of the other. No function can have x-axis symmetry, this checking an equation for such symmetry is another test like the vertical line test to check if an equation does represent a function.

Y-Axis Symmetry

Y-axis symmetry, also known as vertical symmetry, occurs when a figure or shape is mirror-symmetric with respect to the y-axis.

This means that if a point \((x, y)\) is part of the shape, then the point \((-x, y)\) must also be part of the shape.

For example, the letter "I" exhibits y-axis symmetry because it looks the same when mirrored along the y-axis.

$f(x)=x^2$ is a good example of y-axis symmetry. We can fold down the y-axis and both sides would meet. We can test algebraicly or graphically by replacing $x$ with $(-x)$ and then checking to see if we get the same graph or equation.

Symmetry about the Origin

Symmetry about the origin, also known as rotational symmetry of order 2, occurs when a figure or shape is mirror-symmetric with respect to both the x-axis and the y-axis, simultaneously.

This means that if a point \((x, y)\) is part of the shape, then the points \((-x, -y)\), \((-x, y)\), and \((x, -y)\) must also be part of the shape.

For example, a circle exhibits symmetry about the origin because it looks the same when rotated 180 degrees (flipped along both axes).

$f(x)=x^3$ is a good example of origin symmetry. We can rotate 180 degrees and we will be back to where we started. We can test algebraicly or graphically by replacing both $y$ and $x$ with $(-y)$ and $(-x)$ and then checking to see if we get the same graph or equation.
