What is an algorithm?

An algorithm is a finite sequence of well-defined steps to preform some task such that

  • each step is a clear instruction that can be done in a finite amount of time
  • the sequence in which the steps are done is clear
  • the process is guaranteed to stop after a finite number of steps have been done

When we say well-defined we mean that their can be no ambugity about what must be done on this step. That is we either add 2 or we don't. When we implement an algorithm in some programming language we have made a basic comptuer program. When we implemnt an algorithm to solve a real world physical problem we have defined a basic math problem. Let's look at an example of implementing an algorithm.
