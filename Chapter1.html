The Data Table

We store data that we collect in a data table. Progames like google sheets and excel are used to create these tables.

Data tables are essential tools in research for organizing and presenting data in a clear and accessible way. They allow researchers and students alike to quickly analyze and draw conclusions from data. Here, we present a sample data table that showcases various attributes of a group of college students. The table has 5 rows since we got 5 responcese to the survey. Each row is an observation, while each column is a variable.
