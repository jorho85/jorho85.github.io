Visual Demo of transformations

Vertical Translation of a Function

Def 1.9 Given a function $y = f(x)$ and a real number $a$ the transformed function $y = g(x) = f(x) + a$ is a vertical translation of the graph of $f$. That is, every point $(x, f(x))$ on the graph of $f$ gets shifted vertically ot the corresponding point $(x, f(x) + a)$ on the graph of $g$.

Horizontal Translation of a Function

Def 1.10 Given a function $y = f(x)$ and a real number $b$ the transformed function $y = h(x) = f(x - b)$ is a vertical translation of the graph of $f$. That is, every point $(x, f(x))$ on the graph of $f$ gets shifted vertically ot the corresponding point $(x + b, f(x))$ on the graph of $h$.

Vertical stretches

So far, we have seen the possible effects of adding a constant value to function’s output. Next, we investigate the effects of multiplication the function’s output by a constant.

Def 1.11 Given a function $y = f(x)$ and real number $c > 0$, the transformed function $y = v(x) = cf(x)$ is a vertical strech of the graph of $f$. Every point $(x, f(x))$ on the graph of $f$ gets streched vertically to the corresponding point $(x, cf(x))$ on the graph $v$.

If $0 < c < 1$, the graph compresses toward the x-axis.

If $c > 1$ then $v$ is a stretch of $f$ away from the x-axis.

Vertical reflections

Def 1.12 Given a function $y = f(x)$ and real number $c < 0$, the transformed function $y = v(x) = cf(x)$ is a reflection of the graph $f$ across the x-axis followed by a vertical stretch by a factor of $|c|$.

Piecewise review

A piecewise function is a function in which more than one formula is used to define the output. Each formula has its own domain, and the domain of the function is the union of all these smaller domains. We notate this idea like this:

$$f(x) = \begin{cases} \text{algebraic formula 1} & \text{domain of forumla 1} \\ \text{algebraic formula 2} & \text{domain of forumla 2} \\ \text{algebraic formula 3} & \text{domain of forumla 3} \\ \end{cases}$$

Piecewise functions can be split into as many pieces as necessary. Each piece behaves differently based on the input function for that interval. Pieces may be single points, lines, or curves.

The Absolute Value funtion

Def 1.12 The absolute value function can be defined as a piecewise funtion:

$$f(x) = \begin{cases} x & x \geq 0 \\ -x & x < 0 \end{cases}$$

The absolute value function is commonly thought of as providing the distance the number is from zero on a number line. Algebraically, for whatever the input value is, the output is the value without regard to sign.

Example of writing an interval in absolute value form

Let's say we want to describe all values x within our including a distance of 4 from the number 5.

The distance from x to 5 can be represented using the absoluate value as $|x - 5|$

This is the same as saying $-4 \leq (x - 5)$ and $(x - 5) \leq 4$, or $1 \leq x$ and $x \leq 9$

So $|x-5| \leq 4$ is equivalent to $1 \leq x \leq 9$

How to solve an absolute value problem

  1. Isolate the absoluate value expression. That is move everyting not inside the absolute value to the other side of the equation.
  2. Use $|A| = B$ to write $A = B$ and $A = -B$. If $B < 0$ you won't have a solution.
  3. Solve both equation for x
  4. If you had an inequatility, make sure you graph your answers and find the interval.

Notice that we don't solve for x while there is still an inequatility on our page. We have to break down our problem into its smaller parts.

An example

For the function $f(x) = |4x + 1| - 7$ find when $f(x) = 4$

Let's substitute $4$ for $f(x)$ to get $4 = |4x + 1| - 7$

Now we isolate the absolute value by adding 7 to both sides to get $11 = |4x + 1|$

Now we eliminate the absolute value by writing two equations $11 = 4x + 1$ and $-11 = 4x + 1$

Now solve both for x to get $x = \frac{10}{4} = 2.5$ and $x = -3$, which are both solutions.

Review and helpful videos

A transformation of a given function $f$ is a process by which the graph may be shifted or stretched to generate a new, related function with fundamentally the same shape. In this section we considered four different ways this can occur: through a horizontal translation (shift), through a reflection across the line $y=0$ (the x-axis), through a vertical scaling (stretch) that multiplies every output of a function by the same constant, and through a vertical translation (shift). Each of these individual processes is itself a transformation, and they may be combined in various ways to create more complicated transformations.
